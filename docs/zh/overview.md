# Kredo —— 一种创造价值、而非攫取价值的经济

## 问题

两种常见模式各带内建的不公。在**食利资本主义**中，有资本者即便不工作也赚得更多：价值是从所有权
地位中*攫取*，而非被创造。在**公社**中，人人平等，却没有增长机制、也不奖励活跃者，于是出现搭便车
问题。Kredo 是**第三条道路**：回报与价值创造成正比；活跃者赚得更多，但没有人一无所有；遭受攻击时
系统会自行防御。

## 两种代币

货币可以转让，信任不能。两种代币正由这一不对称而生。

- **V（贡献）**——工作货币*兼*俱乐部份额。用于支付服务，赋予外部利润的份额，可通过基金兑换为真实
  货币（USDC）。余额可为负——这即是来自系统的贷款。V 可**转让**。
- **R（声誉）**——灵魂绑定。不可转让、出售或赠送，只能通过工作赚取、因违规而销毁。它提高你的信用
  额度并赋予投票权重（按 $\sqrt{R}$）。

若声誉可被购买，富有者便能在不创造价值的情况下把持系统。因其不可转让，影响力随被证明的贡献而
来，而非钱包的大小。

## 货币如何诞生

一个没有 V 的新人也能获得服务。货币在交易时被创造，且不使任何人贬值：

1. **赊账交易。** B 向 A 订购服务但资金不足；系统为 A 增发 V，B 承担债务（余额为负）。
2. **给所有人的礼物。** 同时为所有活跃成员增发少量 V，以免稀释其积蓄；创造价值越多者得到越多。
3. **礼物在托管中等待。** 冻结至债务偿还：偿还 → 向所有人释放；90 天未还 → 销毁。无人亏空，亦无通胀。
4. **自动刹车。** 礼物规模（$\varepsilon$）随违约增多而下降。

## 自我防御与价格

三种威胁，三种内建应对：**动态 $\varepsilon$**（抗通胀）、**带折扣的赎回队列**（抗恐慌）、
**违约即销毁**（抗欺诈）。

V 的价格为 $P=(\text{基金}+12\times\text{外部收入})/\text{V 总量}$。基金操作对价格中性（已证明）。
**没有外部收入，V 的价格注定下跌**——这不是缺陷，而是数学的必然。

其哲学被形式化为**五条公理**与**七个不变式**，实现为确定性模拟器并接受压力测试。在 **315 次运行**
中——蒙特卡洛、192 点参数扫描、组合压力测试与状态分析——**经济从未崩溃**。细节与协议见
