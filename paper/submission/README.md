# Публикация статьи — пакет для отправки

Эта папка — всё, что нужно, чтобы опубликовать препринт на трёх площадках.
В каждой подпапке два файла:

- `instructions.ru.md` — пошаговая инструкция по-русски (что нажимать);
- `metadata.en.md` — готовые тексты на английском для копипаста в формы.

## Рекомендуемый порядок

| Шаг | Площадка | Зачем | Барьер |
|---|---|---|---|
| 1 | [`zenodo/`](./zenodo) | Мгновенный **DOI** (постоянный идентификатор для цитирования) + архив кода | нет |
| 2 | [`ssrn/`](./ssrn) | Профильная площадка по экономике, где статью найдут экономисты | модерация 1–3 дня |
| 3 | [`arxiv/`](./arxiv) | Самая престижная площадка препринтов | возможен эндорсмент (поручительство) |

Порядок именно такой: Zenodo даёт DOI сразу и без условий; SSRN — простая
модерация; arXiv может попросить поручительство, и если попросит — вы ничего
не теряете, потому что DOI уже есть.

## Что публикуем

- Везде основной файл — **английская версия**: `paper/main.pdf`.
- Русскую версию `paper/main-ru.pdf` прикладываем **только на Zenodo**
  (дополнительным файлом) — arXiv и SSRN обычно одноязычные.
- На arXiv загружается **LaTeX-исходник** (`main.tex`), а не PDF — так у них принято.

## Общий чек-лист перед стартом

- [ ] `main.pdf` и `main-ru.pdf` собраны из последней версии (`make all` в `paper/`)
- [ ] Рабочий email (он будет публичным контактом автора)
- [ ] По желанию: завести [ORCID](https://orcid.org) — «паспорт учёного» за 2 минуты,
      его поле есть на всех трёх площадках (необязательно, но полезно)

## После публикации

- Добавьте полученный DOI (Zenodo) в README репозитория и в поле
  «Comments» на arXiv/SSRN — так все версии свяжутся между собой.
- Если захочется, я добавлю на сайт kredoclub.com кнопку «Read the paper»
  со ссылкой на DOI.
